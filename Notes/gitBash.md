# Git Bash

These are some helpful commands for Git Bash:
- $git remote -v
    - This calls an existing remote repository
- $git status
    - This checks the status of the repository and show if there are any files in the staging area that need to be added and committed to the repository
-$git commit .
    - This will commit all files in the current folder
- $git add "file name here"
    - This will add a specified file to the repository.
